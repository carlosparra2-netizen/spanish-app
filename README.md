# ¡Español! — Spanish Practice App (Streamlit)

A Spanish practice app for 7th grade intermediate learners.

## Modules
- 🃏 Vocabulario — 8 topic sets, 64 flashcards
- ⚡ Gramática — 15 multiple choice questions with feedback
- 📖 Lectura — Reading passages with comprehension questions
- 💬 Conversación — Speaking prompts with 60s timer
- ✍️ Escritura AAPPL — 31 writing prompts with 10-min timer and tips

## Deploy to Streamlit Cloud
1. Fork or upload this repo to GitHub
2. Go to share.streamlit.io
3. Connect your GitHub repo
4. Set main file: `app.py`
5. Deploy!
